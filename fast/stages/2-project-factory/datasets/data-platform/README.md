# Data Platform Dataset

This dataset configures a Data Platform environment using the Project Factory.

It includes:
-   **Projects**: `core-0`, `product-0`, `shared-0`.
-   **VPCs**: Project-local VPCs defined in `vpcs/`.
-   **Tags**: Resource Manager tags for data governance.
-   **Taxonomies**: Data Catalog taxonomies.
-   **Automation**: Service accounts and buckets for automation.

This dataset was ported from the standalone `2-data-platform` stage.
